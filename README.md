# Populook

Search population by city or state.

![](https://github.com/weiying-chen/populook-react/blob/main/screenshot.png)

**[Live Demo](https://populook-react.vercel.app/)**

## Installation

```bash
# Install dependencies
npm install
```

## Usage

```bash
# Run app
npm run dev

# Build app
npm run build
```
